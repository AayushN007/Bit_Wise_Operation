# 🔢 Bitwise Operators Program (Python)

A simple Python program to demonstrate the working of **bitwise operators**.

This project performs different bit-level operations on two numbers and displays the results.

## 📌 About The Project

The program uses two values:

```

a = 5
b = 4

````

and applies:

- Addition
- OR operation
- XOR operation
- Left shift
- Right shift

## ✨ Concepts Covered

✅ Arithmetic operators  
✅ Bitwise OR (`|`)  
✅ Bitwise XOR (`^`)  
✅ Left Shift (`<<`)  
✅ Right Shift (`>>`)  
✅ Binary number operations  

## 🛠️ Technology Used

- Python 3.x

## ▶️ Run The Program

```bash
python bitwise.py
````

## 🖥️ Sample Output

```
9
5
1
10
2
```

## 🧠 Example

Binary representation:

```
5 = 101
4 = 100
```

Operations are performed bit by bit to get the result.
